Computer Systems

Input Devices

  • Keyboard - Text and command input
  • Mouse - Movement and selection
  • Microphone - Audio input
  • Camera - Image and video input
  • Scanner - Document digitization
  • Sensors - Environmental data

Processing

  • CPU (Central Processing Unit)
  • RAM (Random Access Memory)
  • Data calculations and manipulation
  • Program execution
  • Decision making

Output Devices

Devices that show or present processed information to the user:

  • Monitor/Display
  • Speakers
  • Printer
  • Projector

The Input Process

Input devices convert physical actions (like pressing keys or moving the mouse) into digital signals that the computer can understand. These signals are then sent to the processor for handling.

The Processing Stage

The CPU processes data using a cycle of Fetch, Decode, and Execute. It retrieves instructions from memory, understands what needs to be done, and performs the required calculations or operations.

The Output Process

After processing, data is converted back into a form that humans can understand. This might be visual (on a screen), audio (through speakers), or physical (like printed documents). The type of output depends on what the user needs.
